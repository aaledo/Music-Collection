Little utility to consult and maintain music bands and musicians.

Initial screen shows 2 entry options : Artists & Instrument Players.

Artists :

A new selection is proposed for the system to filter the artists : All, by musical style or by name.
Once the choice is made, selected artists rise in new window and new options are available for any of them :
* Maintain relationships : link or unlink other artists.
* Assign instrument players.
* Delete artist.
* Create artist.

Instrument Players :
Subscreen to create new ones, so that they'll be available for artists to link with.
